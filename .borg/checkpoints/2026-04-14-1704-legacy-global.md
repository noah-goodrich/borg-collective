## Decisions Made

- **Additive-only merge** ‚Äî `jq` computes `($live + $new | unique)` rather than overwriting; machine-local allow-entries survive `borg setup` runs across any machine
- **`__DOTFILES_DIR__` substituted by `sed` before `jq`** ‚Äî ensures the marketplace plugin path resolves before JSON is parsed, avoiding a literal placeholder in the merged output
- **Machine-local fields (`model`, `enabledPlugins`) stay out of the versioned base** ‚Äî kept in `~/.config/borg/{claude,cortex}-settings.local.json` overlay templates, not the dotfiles base
- **Template generated once, never overwritten** ‚Äî `if [[ ! -f "$_claude_local" ]]` guard prevents clobbering existing machine customizations
