## 4. Blockers

None. But one mistake is worth carrying forward:

**I misdiagnosed the 127s and shipped the wrong root cause in #69.** I asserted `notifyd`/`cortex-wake` died because
their hardcoded `PATH` omitted `$HOME/.local/bin`. Deploying that fix left both at 127. The actual cause was in their
stderr logs the whole time â€” I theorized instead of reading them:

    lib/borg-hooks.sh:158: no such file or directory: /reaper.sh

`lib/borg-hooks.sh` did `source "$(dirname "${BASH_SOURCE[0]}")/reaper.sh"`, but it is sourced by bash hooks **and** by
two zsh binaries, where `BASH_SOURCE` does not exist. Fixed with `${BASH_SOURCE[0]:-$0}`. Then `cortex-wake` still
exited 1: `(( DEBUG ))` as the last command in `log()` returns 1 under `set -e`. Both fixed in #71, both now pinned by
class-level tripwires in `tests/agent_path.bats`.

**Read the stderr log before theorizing.** That is the lesson, and it cost a merged PR with a false claim in it.
