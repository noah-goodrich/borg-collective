## Decisions Made

- **Additive-only merge** — `borg setup` never removes machine-local `permissions.allow` entries; dotfiles base is a floor, not a ceiling. Avoids breaking machine-specific tool grants across machines.
- **`__DOTFILES_DIR__` substitution** — allows the base settings file to reference dotfiles-relative paths portably before the jq merge runs.
- **Portfolio pivot to Supabase + Fly.io** (the directive) — dropped Snowflake Postgres + SPCS for both Waypoint and wallpaper-kit MVPs based on a Cortex evaluation. Rationale: Snowflake is ~$10/mo loyalty tax with no user benefit at Stage 1; SPCS has wrong shape for bursty consumer traffic; Supabase free tier handles 0–500 households at $0. Two separate Supabase projects (not shared) to keep apps independently sellable.
