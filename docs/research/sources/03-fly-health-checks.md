## Verified Quote(s)

> "Machine health checks run only during deploys and are supported for apps using the rolling and canary deployment strategies ... if a Machine check fails, the deployment will be stopped."

> "your Machines won't automatically restart or stop due to failing their health checks, this needs to be done manually."
