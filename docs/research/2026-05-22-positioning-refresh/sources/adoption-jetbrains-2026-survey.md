## Key Findings

- Claude Code workplace adoption grew 3% → 18% (6×) from Apr–Jun 2025 to January 2026.
- Awareness of Claude Code: 31% → 57% over the same period.
- Claude Code and Cursor tied at 18% workplace adoption, but Claude Code wins satisfaction
  differential 46% "most loved" vs Cursor's 19%.
- Multi-tool usage is dominant among senior developers — they are more likely to run Claude
  Code AND another assistant simultaneously, not pick one.
- Implication for borg: the "manages multiple AI coding tools across multiple projects"
  archetype is a real, growing, senior-skewing segment, not a niche-of-one.
